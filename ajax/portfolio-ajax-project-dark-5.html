Desktop Implementation

Project Info:

This project focuses on developing a new design or redesign for creating a case on Dr. Hud's website. The entire design was to be revamped, including colors, buttons, layout placement, etc. This was done because the mobile version has been prioritized, but they have observed that users log in on desktop but do not complete a case on desktop, it was lacking a “feeling” and did not look like the mobile version.

Goals

The goal is to produce a new desktop design that focuses on usability and is implemented in production.

Project development

– This project lasted for a month and a half, where phase one involved a significant portion of methodology and qualitative research approach. Phase one included methods like:

  • Usability test/interview
  • Competitor analys
  • WCAG analys
  • Affinity mapping
  • Personas
  • Impact/effort mapping

– In phase two, I was able to develop wireframes for testing, and then prototype them. The project was divided into three weeks for phase one and three weeks for phase two, as the design evolves based on feedback from the product owner over time.
